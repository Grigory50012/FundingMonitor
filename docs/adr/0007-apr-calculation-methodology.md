# ADR 0007: Методология расчёта APR и агрегации статистики

**Статус**: Принято  
**Дата**: 2026-06-14

## Контекст

APR (Annual Percentage Rate) — ключевая метрика для арбитража. Нужно единообразно считать:
- Текущий APR по одной ставке
- Исторический APR за периоды (1д, 2д, 3д, 7д, 14д, 21д, 30д)
- Агрегаты: среднее, сумма, StdDev

Биржи имеют разные интервалы выплат: 8ч (3 раза/день), 4ч (6 раз), 1ч (24 раза).

## Решение

### Текущий APR (на одну ставку)

```csharp
// CurrentFundingRate entity
public int NumberOfPaymentsPerDay => FundingIntervalHours > 0 ? 24 / FundingIntervalHours : 3;

public decimal APR => FundingIntervalHours > 0
    ? FundingRate * 100m * (365m * 24m / FundingIntervalHours)
    : 0m;
```

Формула: `FundingRate * (365 * 24 / IntervalHours) * 100%`

Пример: rate=0.0001, interval=8h → `0.0001 * (365 * 3) * 100 = 10.95% APR`

### Исторический APR за период (AprStatsService)

1. Берём все `HistoricalFundingRate` за период для символа/биржи
2. Группируем по выплатам (уже по одной записи на FundingTime)
3. Считаем:
   - `TotalFundingRatePercent = Sum(FundingRate) * 100`
   - `PaymentsCount = Count()`
   - `AvgFundingRatePercent = TotalFundingRatePercent / PaymentsCount`
   - `Apr = AvgFundingRatePercent * PaymentsPerDay * 365`
   - `StdDev` — стандартное отклонение по `FundingRate * 100`

Кэшируется на 5 минут (`IMemoryCache`) по ключу `(symbol, exchange, periodDays)`.

## Последствия

✅ **Плюсы**:
- Единая формула для всех бирж
- Учитывает реальный интервал выплат каждой биржи
- StdDev даёт понимание волатильности
- Кэш снижает нагрузку на БД

❌ **Минусы**:
- APR — экстраполяция, не гарантия будущей доходности
- Не учитывает комиссии, спреды, фандинг-феллы
- StdDev на маленькой выборке (1 день = 3 точки) малоинформативен

## Альтернативы

| Вариант | Причина отказа |
|---------|----------------|
| Просто `FundingRate * 365 * 3 * 100` (фиксированный 8ч) | Не работает для Bybit/OKX с 4ч/1ч интервалами |
| APY (с капитализацией) | Фандинг не реинвестируется автоматически, APR стандарт де-факто |
| Взвешенное по времени (time-weighted) | Выплаты дискретные, равномерные — простое среднее корректно |

## Примечание

Для арбитража сравниваем APR между биржами. Разница APR = ожидаемая годовая доходность стратегии (до комиссий).
