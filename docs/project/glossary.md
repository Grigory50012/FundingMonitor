# Глоссарий

## Funding Rate

Периодическая ставка платежа между long- и short-позициями на рынках perpetual futures.

## APR

Годовая процентная ставка, рассчитанная из периодических funding rates.

## Арбитражная возможность

Разница между funding rates на разных биржах, которую стоит проверить как потенциальную возможность.

## Exchange Client

Инфраструктурный компонент, который получает funding data из API конкретной биржи.
