# Описание проекта

FundingMonitor отслеживает funding rates на криптовалютных биржах и помогает находить возможности для funding arbitrage.

## Цели

- Собирать текущие и исторические funding rates с поддерживаемых бирж.
- Сравнивать ставки между биржами.
- Показывать арбитражные возможности в понятном дашборде.
- Вести архитектуру и решения по разработке рядом с кодом.

## Область

Текущая область:

- Backend API на .NET.
- Хранение данных в PostgreSQL.
- Очередь исторического сбора данных на Redis.
- Frontend на React/Vite.
- Документация в Markdown, управляемая через Obsidian.

Вне области, пока не будет отдельного плана:

- Пользовательские аккаунты.
- Алерты и уведомления.
- Автоматическая торговля.
- Публичный SaaS deployment.
